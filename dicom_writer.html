class DicomWriter¶

Subclassed by DicomWriter2D, DicomWriter3D, DicomWriter3DSliceXY, DicomWriter3DSliceXZ, DicomWriter3DSliceYZ

Public Functions

void saveImage(std::string filepath, gdcm::File &oldFile, double *gamma, int xNumber, int yNumber, int zNumber, int precision, double fillValue, int rewriteTagsStrategyId, std::set<std::pair<int, int>> tags = (std::set<std::pair<int, int>>()), int slice = -1)¶

Saves Gamma Index array and all necessary tags to a DICOM file.

Note

Values of parameters: dims, refXNumber, refYNumber, refZNumber, plane, refSlice should be the same as those passed to calculateGamma() function.

Parameters

  • filepath – Path where the output DICOM file will be saved.

  • oldFile – Reference to the object in which GDCM library stored parsed reference DICOM file. The reference returned by loadDicom() function. Used while transferring tags to the output DICOM file.

  • gamma – The calculated gamma array.

  • xNumber – Number of voxels along x axis in the gamma image.

  • yNumber – Number of voxels along y axis in the gamma image.

  • zNumber – Number of voxels along z axis in the gamma image.

  • precision – Number of digits after the decimal point in the resulting Gamma Index.

  • fillValue – Value that will be assigned to voxels for which the GI comparison was not performed.

  • rewriteTagsStrategyId – Specifies the strategy of transferring tags from reference DICOM to output DICOM file:

    • 5 - rewrite all tags, except these specified in tags parameter

    • 4 - rewrite only these specified in tags parameter

    • 3 - rewrite only not private tags

    • 2 - rewrite no tags

    • 1 or other integer values - rewrite all tags

  • tags – (optional) Required only rewriteTagsStrategy dims is greater than 3. The set specifying which exactly tags (not) to transfer. For more information on how to construct this set see applyDicomTagsRewriteStrategy().

  • slice – (optional) Required only when GI comparison was performed in 2.5D od 2dfrom3D version. Specifies for which slice (in the given orientation) the GI comparison was performed. First slice has number 0.
