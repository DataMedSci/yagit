class DicomReader¶

Public Functions

std::vector<double> acquireImage(gdcm::File &file, int &dims, double &xStart, double &xSpacing, int &xNumber, double &yStart, double &ySpacing, int &yNumber, double &zStart, double &zSpacing, int &zNumber)¶

Retrieves the image matrix and its parameters from DICOM file.

Note

Every parameter, except for the first one , will be filled by the function with the image spatial parameters. Their initial values are not important and will be lost.

Parameters

  • file – Reference to the object in which GDCM library stores parsed DICOM file. The reference returned by loadDicom() function.

  • dims – Number of dimensions of acquired image.

  • xStart – x coordinate of the center of the most left voxel in the image.

  • xSpacing – Distance between centers of adjacent voxels along x axis in the image.

  • xNumber – Number of voxels along x axis in the image.

  • yStart – y coordinate of the center of the most top voxel in the image.

  • ySpacing – Distance between centers of adjacent voxels along y axis in the image.

  • yNumber – Number of voxels along y axis in the image.

  • zStart – z coordinate of the center of the most front voxel in the image.

  • zSpacing – Distance between centers of adjacent voxels along z axis in the image.

  • zNumber – Number of voxels along z axis in the image.

Returns

Pointer to the image matrix. Its size can be calculated using parameters: dims, xNumber, yNumber and zNumber.
