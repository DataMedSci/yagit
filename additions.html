bool *convertArrayToBinaryMask(int size, double *array, bool ((*converter)(double)) = NULL)¶

Converts a double array to binary mask.

Using custom converter function, constructs a binary mask with the size of input array.

If no converter is specified (or NULL is passed) the default converter is applied.

Default converter returns True if array value is greater than 0.

Note

Returned array has to be freed by the user.

Parameters

  • size – The length of linearized array representing array used to construct binary mask.

  • array – Linearized double array representing array used to construct binary mask.

  • converter – (optional) Pointer to function determining whether given value in original mask should evaluate to True or False.

Returns

Pointer to binary mask with the size of given array.

void applyMaskToImage(int size, double *image, bool *mask)¶

Image and mask must be the same size.

void applyNoiseFilteringToImage(int size, double *image, double noisePercentageLimit)¶

Eliminates from the image values considered as a noise.

Note

The threshold will be calculated using formula: threshold = minVal + (maxVal - minVal) * noisePercentage %

Parameters

  • size – The length of linearized array representing the image.

  • image – Linearized double array representing image.

  • noisePercentage – Used to calculate threshold below which the voxels in the image will be considered as a noise.

void applyLinearMappingToImage(int size, double *image, double multiply, double add)¶

Modifies image using a linear function.

Each element x in the image is set as follows: x <- multiply * x + add

Parameters

  • size – The length of linearized array representing the image.

  • image – Linearized double array representing image.

  • multiply – See above.

  • add – See above.

int calculateGammaArraySize(int dims, int refXNumber, int refYNumber = 1, int refZNumber = 1, int plane = -1)¶

Calculates the Gamma Index array size. It is usually the same as reference image size, but in case of 2.5D and 2Dfrom3D comparisons it differs.

Note

All parameters should be the same as those passed to calculateGamma() function.

Parameters

  • dims – Number of dimensions of the images under comparison:

    • 1 - comparison 1D,

    • 2 - comparison 2D,

    • 3 - comparison 3D,

    • 4 - comparison 2.5D (comparison of one slice of the reference image against the whole target image),

    • 5 - comparison 2Dfrom3D (comparison of one slice of the reference image against another slice of the target image).

  • refXNumber – Number of voxels along x axis in the reference image.

  • refYNumber – Number of voxels along y axis in the reference image.

  • refZNumber – Number of voxels along z axis in the reference image.

  • plane – (optional) Required only when dims is greater than 3. Specifies the orientation of the reference slice:

    • 0 - parallel to XY plane,

    • 1 - parallel to XZ plane,

    • 2 - parallel to YZ plane.

Returns

Size of Gamma Index array.

double gammaPassingRate(int size, double *gamma)¶

Calculates Gamma Passing Rate.

Gamma Passing Rate is the percentage of voxels for which the Gamma Index value is smaller than 1.0.

Parameters

  • size – The length of linearized array representing image of GI values. It can be calculated using calculateGammaArraySize()

  • gamma – Linearized double array representing image of GI values.

Returns

Gamma Passing Rate.

std::string createComputationDescription(std::string refFile, std::string tarFile, int algorithm)¶

Constructs brief example of description of the comparison. Can be used to distinguish entries in statistics file.

Note

One can specify their own description and pass it to the: prepareStatisticsFile() function.

Parameters

  • refFile – Name of the reference file.

  • tarFile – Name of the target file.

  • algorithm – Algorithm used for comparison.

    • 0 - Classic Solver,

    • 1 - Spiral Solver with rectangle,

    • 2 - Spiral Solver without rectangle.

Returns

A short description.

  • filepath – Path to the file where the statistics will be saved (default: stats.txt).

int *prepareHistogram(int gammaSize, double *gamma, int delimitersSize, double *delimiters)¶

Creates a histogram of Gamma Index values.

Constructed histogram is an array of integers, in which i-th element states for the number of voxels in gamma image that have value in the range <delimiters[i]; delimiters[i+1]).

Note

Returned array has to be freed by the user.

Parameters

  • gammaSize – The length of linearized array representing image of GI values.

  • gamma – Linearized double array representing image of GI values.

  • delimitersSize – The length of the delimiters array.

  • delimiters – Pointer to double array, which values are the boundaries of the consecutive brackets of the histogram.

Returns

Array of integers representing histogram. The array size is equal to delimiters-1.

int *prepareHistogram(int gammaSize, double *gamma, double xMin, double xMax, int bucketsNumber)¶

Creates a histogram of Gamma Index values.

Constructed histogram is an array of integers. Histogram consists of bucketsNumber buckets of the same size (step = (xMax - xMin)/bucketsNumber)). The i-th element of the histogram states for the number of voxels in gamma image that have value in the range <xMin + i * step; xMin + (i+1) * step).

Note

Returned array has to be freed by the user.

Parameters

  • gammaSize – The length of linearized array representing image of GI values.

  • gamma – Linearized double array representing image of GI values.

  • xMin – Minimum value that is taken into the histogram.

  • xMax – Maximum value that is taken into the histogram.

  • bucketsNumber – Number of buckets that the range <xMin; xMax) is divided into.

Returns

Array of integers representing histogram. The array size is equal to bucketsNumber.
