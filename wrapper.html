gdcm::File &loadDicom(std::string filepath)¶

Loads DICOM file into an GDCM object.

Note

The returned object should be later passed to other functions from this module.

Parameters

filepath – Path to DICOM file tha should be loaded.

Returns

An GDCM object containing parsed DICOM file.

std::vector<double> acquireImage(gdcm::File &file, int &dims, double &xStart, double &xSpacing, int &xNumber, double &yStart, double &ySpacing, int &yNumber, double &zStart, double &zSpacing, int &zNumber)¶

Retrieves the image matrix as linearized array and its parameters from DICOM file.

Returned array must be freed by the user.

Note

Every parameter, except for the first one - dataSet, will be filled by the function with the image spatial parameters. Their initial values are not important and will be lost.

Parameters

  • file – Reference to the object in which GDCM library stores parsed DICOM file. The reference returned by loadDicom() function.

  • dims – Number of dimensions of of acquired image.

  • xStart – x coordinate of the center of the most left voxel in the image.

  • xSpacing – Distance between centers of adjacent voxels along x axis in the image.

  • xNumber – Number of voxels along x axis in the image.

  • yStart – y coordinate of the center of the most top voxel in the image.

  • ySpacing – Distance between centers of adjacent voxels along y axis in the image.

  • yNumber – Number of voxels along y axis in the image.

  • zStart – z coordinate of the center of the most front voxel in the image.

  • zSpacing – Distance between centers of adjacent voxels along z axis in the image.

  • zNumber – Number of voxels along z axis in the image.

Returns

Pointer to the image array. It size can be calculated using parameters: dims, xNumber, yNumber and zNumber.

void saveImage(int dims, gdcm::File &oldFile, std::string filepath, double *gamma, int refXNumber, int refYNumber, int refZNumber, int precision, double fillValue, int rewriteTagsStrategy = 1, const std::set<std::pair<int, int>> tags = (std::set<std::pair<int, int>>()), int plane = -1, int refSlice = -1)¶

Saves Gamma Index array and all necessary tags to a DICOM file.

Note

Values of parameters: dims, refXNumber, refYNumber, refZNumber, plane, refSlice should be the same as those passed to calculateGamma() function.

Parameters

  • dims – Number of dimensions of the images under comparison:

    • 1 - comparison 1D,

    • 2 - comparison 2D,

    • 3 - comparison 3D,

    • 4 - comparison 2.5D (comparison of one slice of the reference image against the whole target image),

    • 5 - comparison 2Dfrom3D (comparison of one slice of the reference image against another slice of the target image).

  • oldFile – Reference to the object in which GDCM library stored parsed reference DICOM file. The pointer returned by loadDicom() function. Used while transferring tags to the output DICOM file.

  • gamma – The calculated gamma array.

  • filepath – Path where the output DICOM file will be saved.

  • refXNumber – Number of voxels along x axis in the gamma image.

  • refYNumber – Number of voxels along y axis in the gamma image.

  • refZNumber – Number of voxels along z axis in the gamma image.

  • precision – Number of digits after the decimal point in the resulting Gamma Index.

  • fillValue – Value that will be assigned to voxels for which the Gamma Index comparison was not performed or had assigned NaN value during or before comparison (e.g. due to noise filtering). We propose to choose the value 1.1*limit or limit+1.

  • rewriteTagsStrategy – Specifies the strategy of transferring tags from reference DICOM to output DICOM file:

    • 5 - rewrite all tags, except these specified in tags parameter,

    • 4 - rewrite only these specified in tags parameter,

    • 3 - rewrite only not private tags,

    • 2 - rewrite no tags,

    • 1 or other integer values - rewrite all tags.

  • tags – The set specifying which exactly tags (not) to transfer. For more information on how to construct this set see applyDicomTagsRewriteStrategy().

  • plane – (optional) Required only when dims is greater than 3. Specifies the orientation of the gamma slice:

    • 0 - parallel to XY plane,

    • 1 - parallel to XZ plane,

    • 2 - parallel to YZ plane.

  • refSlice – (optional) Required only when dims is greater than 3. Specifies for which reference slice (in the given orientation) the GI comparison was performed. First slice has number 0.
